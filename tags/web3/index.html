Twitter using NFT avatar

准备工作 OpenSea 帐号 MetaMask 钱包 Coinbase 钱包 开通步骤 开通 Twitter Blue 推荐使用iPhone打开Twitter官方客户端，我的Apple ID是美区的，可以直接通过内购开通，价格为每月 4.99USD。 往你的 MetaMask 钱包充钱 正常情况下，等值2USD的ETH足够了。因为我是用OKX向我的MetaMask转账的，OKX的最低转账是0.01ETH，所以我充值了大概130CNY全部买成ETH。（0.01ETH大概可以生成6-7个NFT头像） 请注意最好是复制粘贴你的 MetaMask 的ETH收款地址，输错无法找回。 以上这笔费用，主要是用于转移你的NFT头像头像的手续费。大概不到2USD。 在OpenSea上传你的头像 点击 Create 按扭。上传你的头像，并且取一个名字即可。 转移你的头像到 Coinbase 点击红色框框内的 转移 图标。输入你的 Coinbase ETH地址，这个时候会需要你支付手续费。你直接支付即可。 更换Twitter头像 在Twitter上点击更换头像，选择NFT，这个时候会提示你连接你的ETH钱包，你连接你的 Coinbase 钱包之后，会发现已经有一个可选的头像了。换好之后就是六边形的NFT专属头像。
