Twitter using NFT avatar

准备工作 OpenSea 帐号 MetaMask 钱包 Coinbase 钱包 开通步骤 开通 Twitter Blue 推荐使用 iPhone 打开 Twitter 官方客户端，我的 Apple ID 是美区的，可以直接通过内购开通，价格为每月 4.99USD。 往你的 MetaMask 钱包充钱 正常情况下，等值 2USD 的 ETH 足够了。因为我是用 OKX 向我的 MetaMask 转账的，OKX 的最低转账是 0.01ETH，所以我充值了大概 130CNY 全部买成 ETH。（0.01ETH 大概可以生成 6-7 个 NFT 头像） 请注意最好是复制粘贴你的 MetaMask 的 ETH 收款地址，输错无法找回。 以上这笔费用，主要是用于转移你的 NFT 头像头像的手续费。大概不到 2USD。 在 OpenSea 上传你的头像 点击 Create 按扭。上传你的头像，并且取一个名字即可。 转移你的头像到 Coinbase 点击红色框框内的 转移 图标。输入你的 Coinbase ETH 地址，这个时候会需要你支付手续费。你直接支付即可。 更换 Twitter 头像 在 Twitter 上点击更换头像，选择 NFT，这个时候会提示你连接你的 ETH 钱包，你连接你的 Coinbase 钱包之后，会发现已经有一个可选的头像了。换好之后就是六边形的 NFT 专属头像。
